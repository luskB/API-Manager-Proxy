## APIManagerProxy v1.7.3

This release focuses on monitoring accuracy and day-to-day usability improvements across the desktop app.

### Highlights

- Synced monitor billing now propagates into Token statistics and dashboard summaries.
  After using **Sync Costs** in the monitor page, the corresponding aggregated cost views now refresh from the synchronized site billing data instead of keeping stale pre-sync estimates.

- Monitoring controls behave more consistently for repeated use.
  Auto-refresh now defaults to enabled when opening the monitor page, and the top action row has been refined so the **Sync Costs** action sits alongside the existing controls without breaking the layout.

- Time-range preferences are remembered across page switches.
  Dashboard and Token statistics now keep your last selected hourly / daily / weekly scope instead of resetting to daily every time you navigate away and back.

### Included fixes

- Improved monitor-to-stats consistency when site billing values differ from local estimates.
- Preserved existing site-billing sync behavior while making downstream totals update correctly.
- Refined the monitor toolbar layout to reduce accidental clicks while keeping common actions close together.

### Privacy and packaging

- This release is built from repository source only.
- Local runtime configuration, browser profiles, logs, caches, and build outputs remain excluded from source control.
- Release artifacts are generated by GitHub Actions from the sanitized repository state.
